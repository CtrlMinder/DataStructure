DataStructure

2.队列

3.链表
3.1链表倒数第K个节点    
3.1.1 （视频算法）通过遍历获取链表长度，二次遍历将指针移动到(length - index + 1)个节点位置 - O(N+N-K)   
3.1.2 （优化）快慢指针 - 快指针先移动k - 1次到第k个节点，快指针与慢指针同时遍历，在快指针遍历结束时，慢指针指向倒数第k个节点 - O(N)
